CyberCash

CyberCash é o projeto de criptomoeda (moeda digital) que busca ter as seguintes características:

  1. segura
  2. eficiente
  3. útil
  4. estável
  5. acessível
  6. rentável

Apenas após um plano focado nesses pontos estiver concluído e estes objetivos puderem comprovadamente ser alcançados é que o projeto será divulgado. As características foram dispostas em ordem de importância no plano de criação.
